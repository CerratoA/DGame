## Technologies Used

This project leverages several technologies and frameworks to bring the Pokémon world to life:

- **React**

- **Vite**

- **Phaser**

- **Bun**

## Play

To launch the game:

```bash
git clone
cd pokemon
bun install
bun dev
```
